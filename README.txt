== Description ==
A great extension for WooCommerce that helps you manually add new orders as a customer fast and easy. 
Be amazing to have a "private" page for admins where admins can create orders easily which supports all the features that the normal checkout page the customers use supports
Give great flexibility and save time to shop owners.
Integrated: Paypal Invoice, Holder Invoice

== Installation ==

1. Upload `woo-manual-orders` folder/directory to the `/wp-content/plugins/` directory
2. Activate the plugin (Wordpress -> Plugins).
3. Go to Woocommerce >> Settings >> Manual Order
4. Choose page where the function to do, or click "auto create page" to create page with shortcode content, also auto assign option value
5. Choose roles which allow access the function.
6. Click "Save changes" to complete install

That's it! You are ready to use this plugin
